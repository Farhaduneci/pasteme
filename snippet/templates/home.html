What is PasteMe?!

PasteMe is a RESTful pastebin service that allows users/developers share their snippets, code blocks, buggy functions through almost all platforms. There is a CLI tool called pasteme-cli that interacts with the open-source APIs and helps you record your snippets from terminals and CLI shells.

pasteme-cli Installation

This CLI tool is an open-source SDK that interacts with the PasteMe APIs. The following steps would help you install this package on your distribution.

$ git clone https://github.com/collove/pasteme-cli.git
$ cd pasteme-cli
$ ./setup.sh

Usage

$ echo "print('hey')" | pasteme -l python
https://pasteme.pythonanywhere.com/paste/a1c13
$ cat main.cpp | pasteme -l cpp
https://pasteme.pythonanywhere.com/paste/c12b4
